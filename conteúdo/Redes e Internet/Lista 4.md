
## 📜 Lista  de Exercícios sobre Rede e Internet

1. O que é um protocolo de comunicação?
2. Quais são os principais protocolos de comunicação utilizados na Internet?
3. O que é um endereço IP?
4. Quantos endereços distintos podem ser representados por um endereço IP no formato IPv4? E no IPv6?
5. O que é um endereço MAC?
6. O que é um servidor DNS?
7. O que é uma rede local (LAN)?
8. Para que serve um roteador?
9. Qual a faixa de endereços IP reservada para redes privadas e qual a função dos endereços `.0` e `.255` em uma sub-rede com máscara `255.255.255.0`?
10. O que é a notação CIDR? Quantos endereços de host úteis existem em uma rede `/24`?
11. Que tipo de informação é transmitida por meio do protocolo HTTP?
12. O que significa HTTPS e qual sua principal vantagem sobre o HTTP?
13. Que comando posso utilizar para copiar um arquivo de um computador para outro via SSH? Dê um exemplo.
14. Como posso verificar se um computador está acessível via rede? Dê um exemplo.
15. Como o SSH usa chave pública para autenticação? Dê um exemplo prático de configuração.
16. Qual porta TCP é utilizada por padrão pelo protocolo SSH?
17. Descreva como criar um túnel SSH para acessar um serviço que está rodando na porta `8080` de um servidor remoto, mas que está bloqueado por firewall externo. Escreva o comando.
18. Qual usuário devo utilizar para acessar um servidor FTP de forma anônima?
19. Como faço para listar os arquivos de um servidor FTP via terminal? Dê um exemplo.
20. Como faço para baixar um arquivo de um servidor FTP para o meu computador? Dê um exemplo.
21. Como faço para baixar múltiplos arquivos binários de um servidor FTP cuja extensão seja `.zip`?
22. Como verificar o endereço IP e a máscara de rede de todas as interfaces ativas usando o comando `ip`?
23. Como descobrir qual é o gateway padrão configurado na sua máquina via terminal?
24. Qual comando moderno pode ser utilizado para visualizar todas as conexões TCP/UDP ativas e as portas em escuta? Dê um exemplo.
25. Como consultar o registro `MX` de um domínio (ex: `gmail.com`) para identificar seus servidores de e-mail?
26. Escreva um comando `curl` que faça uma requisição `GET` para `http://example.com` e exiba apenas os cabeçalhos HTTP e o código de status.
27. Como verificar a rota (saltos) que os pacotes percorrem até um destino remoto (ex: `8.8.8.8`)?
28. Utilizando `netcat` (`nc`), como testar se a porta `443` de um servidor remoto está aberta?
29. Como descobrir o seu endereço IP público diretamente do terminal, sem abrir navegador?
30. Escreva um script Bash simples que teste a conectividade com três hosts (`google.com`, `github.com`, `192.168.1.1`) e imprima `ALIVE` ou `DOWN` para cada um.
